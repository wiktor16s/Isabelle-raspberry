# Isabelle

## Build and start
*`npm install --global --production windows-build-tools` for Windows

`yarn install`

### production:

`yarn start`

### development:

`yarn start:watch`

`yarn build:watch`

### commands to say:

`isabelle` to start capture commands. Its like "OK Google".
 
 `(która) godzina`
 
 `data`
 
 `jak masz na imię`
 
 `mam na imię (imie)`
 
 `(powiedz) dowcip`
 
 `jak mam na imię`

`(to wszystko) dziękuję` to stop capture commands
